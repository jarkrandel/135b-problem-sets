    Problem Set 5 Solutions
    Problem 1: You’ve accidentally introduced a single high-velocity neutron into a lump of Uranium-235 that begins a branching process inside the lump. At each step of the process, every high-velocity neutron impacts a U235 nucleus resulting in one of three possibilities:
    🔗
    1. Absorption: The neutron is absorbed by the nucleus, producing no offspring neutrons.
      🔗
      🔗
    2. Elastic Scattering: The neutron strikes the nucleus and rebounds with high-velocity, surviving to the next stage of the process.
      🔗
      🔗
    3. Binary fission: The neutron breaks apart the nucleus and produces two new high-velocity neutrons that take its place.
      🔗
      🔗
    Suppose that binary fission and elastic scattering occur with probability \(p/2\text{,}\) and absorption occurs with probability \(1-p\text{.}\) Let \(V_n\) denote the random number of high-velocity neutrons at the \(n\)-th step of this process.
    🔗
    1. If \(V_n\) grows without bound over time, then we say a chain reaction has occurred. Let \(c_0(p)\) be the probability that a chain reaction occurs for a fixed value of \(p\in [0,1]\text{.}\) Determine \(c_0(p)\) for all values of \(p\in [0,1]\text{.}\)
      🔗
      🔗
    2. Compute \(\mathrm{E}(V_n)\) as a function of \(p\text{.}\)
      🔗
      🔗
    3. Write an expression for the probability that \(V_2 = 2\) and the process dies out by generation 5. (You do not have to compute explicit numerical values for complicated expressions as long as they are computable in terms of known functions/quantities).
      🔗
      🔗
    4. Thankfully, the first branching process died out. However, in your carelessness, you have now somehow simultaneously introduced a large number \(M {\gt} 0\) of high-velocity neutrons into the lump. Compute the probability \(c_0(M,p)\) that a chain reaction occurs as a function of \(p\) and \(M\text{.}\)
      🔗
      🔗
    5. Under the same setup as in (d), compute \(\mathrm{E}(V_n)\) as a function of \(p\) and \(M\text{.}\)
      🔗
      🔗
    Solution:
    🔗
    1. A chain reaction occurs if and only if the branching process does not die out, which happens with probability \(c_0(p) = 1-\pi_0(p)\) since \(\pi_0\) is the probability that the process dies out. Recall that we can compute \(\pi_0\) by finding the smallest positive number \(s\in [0,1]\) that is a solution to the equation \(\phi (s) = s\) where \(\phi (s) = \sum_{k=1}^\infty p_ks^k\) is the moment generating function of the offspring distribution. In this case, the offspring distribution is
      🔗
      \begin{equation*} p_0 = 1-p,\ p_1 = \frac{p}{2},\ p_2 = \frac{p}{2},\ p_i = 0 \text{ for all } i\not\in \{ 0,2\}. \end{equation*}
      This means that
      🔗
      \begin{equation*} \phi (s) = 1-p + \frac{p}{2}s + \frac{p}{2}s^2, \end{equation*}
      so we are looking for the smallest solution to the equation
      🔗
      \begin{equation*} s = 1-p + \frac{p}{2}s + \frac{p}{2}s^2 \iff ps^2 + (p-2)s + 2(1-p) = 0. \end{equation*}
      There are many ways to solve this. Using the fact that \(s = 1\) is always a solution, we can just write
      🔗
      \begin{equation*} ps^2 + (p-2)s + 2(1-p) = (s-1)(as+b) = as^2 + (a-b)s - b \end{equation*}
      and immediately read off \(a = p\text{,}\) \(b = -2(1-p)\) so that the solution other than \(1\) is a solution to
      🔗
      \begin{equation*} 0 = as+b \iff 0 = ps-2(1-p) \iff s = \frac{2(1-p)}{p}. \end{equation*}
      Notice that
      🔗
      \begin{equation*} \frac{2(1-p)}{p} {\gt} 1 \iff p {\lt} \frac{2}{3}, \end{equation*}
      so when \(p \leq \frac{2}{3}\text{,}\) we know that \(s = 1\) is actually the smallest solution in \([0,1]\text{,}\) and \(\pi_0(p) = 1\) and \(c_0(p) = 0\text{.}\) Otherwise, \(\pi_0(p) = \frac{2(1-p)}{p}\) and \(c_0(p) = 1-\frac{2(1-p)}{p} = \frac{3p-2}{p}\text{.}\) We’ve concluded that
      🔗
      \begin{equation*} c_0(p) = \begin{cases} 0 & \text{if $p \leq \frac{2}{3}$}\\ \frac{3p-2}{p} & \text{if $p {\gt} \frac{2}{3}$.} \end{cases} \end{equation*}
      🔗
    2. Notice that \(\mathrm{E}(X_1) = 0(1-p) + 1\cdot \frac{p}{2} + 2\cdot \frac{p}{2} = \frac{3}{2}p\) so that
      🔗
      \begin{equation*} \mu^n = \mathrm{E}(V_n) = \bigg(\frac{3}{2}p\bigg)^n. \end{equation*}
      🔗
    3. We can use conditional probability to break this problem into stages. Notice that
      🔗
      \begin{equation*} \mathrm{P}(V_2=2\text{ and }V_5 = 0) = \mathrm{P}(V_5 = 0|V_2=2)\mathrm{P}(V_2=2). \end{equation*}
      We can find \(\mathrm{P}(V_2=2)\) by imagining the possible numbers of neutrons produced in generations 1 and 2 that lead to \(V_2=2\text{.}\) It must be that \(V_1 \in \{ 1,2\} \) or else the population dies out at generation 1. Therefore, we can write
      🔗
      \begin{equation*} \mathrm{P}(V_2 = 2) = \mathrm{P}(V_2=2|V_1=1)\mathrm{P}(V_1=1) + \mathrm{P}(V_2=2|V_1=2)\mathrm{P}(V_2=2). \end{equation*}
      If \(V_1 = 1\text{,}\) then this single neutron must produce to two new neutrons to achieve \(V_2 = 2\text{,}\) meaning \(\mathrm{P}(V_2=2|V_1=1) = p/2\text{.}\) On the other hand, if \(V_1 = 2\text{,}\) then either both individuals give birth to a single individual or one gives birth to two and the other gives birth to zero. There are two ways of this final scenario occurring, so \(\mathrm{P}(V_2=2|V_1=2) = (p/2)^2 + 2(p/2)(1-p)\text{.}\) Therefore, using our previous conditional probability equation,
      🔗
      \begin{equation*} \mathrm{P}(V_2=2) = (p/2)(p/2) + ((p/2)^2 + p(1-p))(p/2)=\frac{3}{4}p^2(1-p). \end{equation*}
      On the other hand, once we know that \(V_2=2\text{,}\) \(V_5 = 0\) if and only if the processes associated to each of the two neutrons in generation 2 die out within the next three generations. The probability that a single one’s process dies out in this time is
      🔗
      \begin{equation*} \delta_3 = \phi (\phi (\phi (0))) \end{equation*}
      where \(\phi \) is the moment generating function of the offspring distribution. Since both have independent family trees, we see that
      🔗
      \begin{equation*} \mathrm{P}(V_5=0|V_2=2) = \phi (\phi (\phi (0)))^2. \end{equation*}
      Therefore, our answer is
      🔗
      \begin{equation*} \mathrm{P}(V_2=2\text{ and }V_5 = 0) = \frac{3}{4}p^2(1-p)\phi (\phi (\phi (0)))^2 \end{equation*}
      🔗
    4. In this situation, we now have \(M\) independent copies of the previous branching process at time 0. This means that a chain reaction occurs if and only if a chain reaction occurs in one of these processes. This does not happen exactly when every process dies out, which happens with probability \(\pi_0(p)^M\text{.}\) So, the chain reaction probability is
      🔗
      \begin{equation*} c_0(p,M) = 1 - \pi_0(p)^M = \begin{cases} 0 & \text{if $p \leq \frac{2}{3}$}\\ 1-\left(\frac{2(1-p)}{p}\right)^M & \text{if $p {\gt} \frac{2}{3}$}. \end{cases} \end{equation*}
      🔗
    5. Each of the \(M\) starting neutrons have independent family trees, and the expected number of individuals in each tree at generation \(n\) is \((\frac{3}{2}p)^n\) by (b). So, the total expected number is just
      🔗
      \begin{equation*} M\bigg(\frac{3}{2}p\bigg)^n \end{equation*}
      🔗
    Problem 2: An online casino allows bettors to simultaneously play as many simulated slot machines as they like. Each machine takes in \(\$ 1\) for each spin and pays out randomly. You have insider knowledge of the payout scheme:
    🔗
    • With probability \(\frac{12}{21}\text{,}\) you win nothing.
      🔗
      🔗
    • With probability \(\frac{7}{21}\text{,}\) you win \(\$ 2\text{.}\)
      🔗
      🔗
    • With probability \(\frac{2}{21}\text{,}\) you win \(\$ 3\text{.}\)
      🔗
      🔗
    You begin with $1 and are an all or nothing player. At stage \(n\text{,}\) you take your current bankroll of \(B_n\) dollars and “reinvest” them by playing \(B_n\) new games at the next iteration. Your goal is to make $1,000,000 or go broke.
    🔗
    1. Compute your expected bankroll \(B_1\) after a single play. (This is how much money you receive when you put $1 in. Don’t count the $1 loss).
      🔗
      🔗
    2. Since you understand probability, you wouldn’t play such a game for any serious number of spins. However, you’ve discovered a vulnerability in the software. By pressing buttons in a particular way in the interface, you can change roughly 1/3 of your losing hands at each stage to \(\$ 3\) winners. Compute your new expected bankroll \(B_1\) with your new edge.
      🔗
      🔗
    3. Suppose you can get through each stage in 1 second. How many seconds does it take for your expected bankroll to exceed \(\$ 1,000,000\text{?}\) (You can use a calculator).
      🔗
      🔗
    4. Compute the probability that you go broke eventually.
      🔗
      Comment on (d): You can use the fact that \(s=1\) is always a solution to your equation to factor the third degree polynomial analytically if you want to do this the old-fashioned way.
      🔗
      🔗
    Solution:
    🔗
    1. Let \(B_1\) be the expected return. We have
      🔗
      \begin{equation*} \mathrm{E}(B_1) = 0\cdot \frac{12}{21} + 2\cdot \frac{7}{21} + 3\cdot \frac{2}{21} = \frac{20}{21}. \end{equation*}
      🔗
    2. In these new circumstances,
      🔗
      \begin{equation*} \begin{aligned} \mathrm{P}(\text{You win \$ 3}) & = \mathrm{P}(\text{You win \$ 3 immediately}) + \mathrm{P}(\text{You win \$ 1 initially, but then succeed in changing it to \$ 3})\\ & = \frac{2}{21} + \frac{12}{21}\cdot \frac{1}{3} =\frac{6}{21} \end{aligned} \end{equation*}
      and
      🔗
      \begin{equation*} \mathrm{P}(\text{You win nothing}) = \frac{12}{21}\cdot \bigg(1 - \frac{1}{3}\bigg) = \frac{8}{21}. \end{equation*}
      🔗
    3. In this case,
      🔗
      \begin{equation*} \mu = E(B_1) = 2\cdot \frac{7}{21} + 3\cdot \frac{6}{21} = \frac{32}{21}. \end{equation*}
      This means that \(\mathrm{E}(B_n) = \mu^n = \left(\frac{32}{21}\right)^n\) so that \(\mathrm{E}(B_n)\) exceeds \(1,000,000\) for the first time when \(n = 33\) and
      🔗
      \begin{equation*} \mathrm{E}(B_{33}) = \left(\frac{32}{21}\right)^{33} = 1,088,209.61... \end{equation*}
      🔗
    4. You go broke if this branching process ever dies out. This happens with probability \(\pi_0\text{,}\) the smallest solution \(s\in [0,1]\) to the equation \(s = \phi (s) = \sum_{k=0}^\infty p_ks^k\text{.}\) In this case, the offspring distribution is
      🔗
      \begin{equation*} p_0 = \frac{8}{21},\ p_2 = \frac{7}{21},\ p_3 = \frac{6}{21},\ p_i = 0 \text{ if $i\not\in \{ 0,2,3\} $}. \end{equation*}
      This means we are looking for the solution to the equation
      🔗
      \begin{equation*} s = \frac{8}{21} + \frac{7}{21}s^2 + \frac{6}{21}s^3 \iff 6s^3+7s^2-21s+8 = 0 \iff (s-1)(2s-1)(3s+8) = 0. \end{equation*}
      The smallest positive solution to this is \(s = \frac{1}{2}\text{,}\) so \(\pi_0 = \frac{1}{2}\text{.}\)
      🔗
      🔗
