    Problem 1: Let \(X\) be a \(\mathrm{Normal}(0,1)\) random variable. We’ve already shown that \(\phi_X(t) = e^{t^2/2}\text{.}\)
    🔗
    1. Let \(\mu , \sigma {\gt} 0\) and let \(Z\) be \(\mathrm{Normal}(0,1)\text{.}\) Show that the random variable \(Y = \mu + \sigma Z\) is \(\mathrm{Normal}(\mu ,\sigma )\) by using the change of variables formula \(f_Y(y) = f_X(x(y))|\frac{dx}{dy}|\) with \(y =\mu + \sigma x.\) Compute the moment generating function of \(Y\text{.}\)
      🔗
      🔗
    2. Now, if \(X\) is \(\mathrm{Normal}(\mu_1,\sigma_1^2)\text{,}\) \(Y\) is \(\mathrm{Normal}(\mu_2,\sigma_2^2)\text{,}\) and \(X\) and \(Y\) are independent, show that \(X+Y\) is also normal. Compute its mean and variance in terms of \(\mu_i\) and \(\sigma_i\text{.}\)
      🔗
      🔗
    Solution:
    🔗
    1. We have \(x(y) = (y-\mu )/\sigma \) so that \(\frac{dx}{dy} = 1/\sigma \text{.}\) Therefore,
      🔗
      \begin{equation*} f_Y(y) = f_X(x)\left|\frac{dx}{dy}\right| = \frac{1}{\sqrt{2\pi }}e^{-\frac{x(y)^2}{2}}\frac{1}{\sigma } = \frac{1}{\sqrt{2\pi \sigma^2}}e^{-\frac{(y-\mu )^2}{2\sigma^2}} \end{equation*}
      which is exactly the pdf of a \(\mathrm{Normal}(\mu ,\sigma^2)\) random variable. Now, let \(Y = \mu + \sigma Z\text{.}\) Part (a) showed that \(Y\) is \(\mathrm{Normal}(\mu ,\sigma^2)\) so that we just need to compute \(\phi_Y(t)\text{.}\) Notice
      🔗
      \begin{equation*} \phi_Y(t) = \mathrm{E}(e^{tY}) = \mathrm{E}(e^{t(\mu + \sigma Z)}) = e^{t\mu }\mathrm{E}(e^{t\sigma Z}) = e^{t\mu }\cdot e^{(t\sigma )^2/2} = \exp (t\mu + t^2\frac{\sigma^2}{2}). \end{equation*}
      🔗
    2. To show that \(X+Y\) is normal, we will show that the MGF of \(X+Y\) is the same as the MGF of a normal variable. Since \(X\) and \(Y\) are independent, Lemma 10.1 implies \(\phi_{X+Y}(t) = \phi_X(t)\phi_Y(t)\text{.}\) Using part (a), we see
      🔗
      \begin{equation*} \phi_{X+Y}(t) = \phi_{X}(t)\phi_Y(t) = \exp (t\mu_1 + t^2\frac{\sigma_1^2}{2})\exp (t\mu_2 + t^2\frac{\sigma_2^2}{2}) = \exp (t(\mu_1+\mu_2) + t^2\frac{(\sigma_1^2+\sigma_2^2)}{2}). \end{equation*}
      Using part (a) again, we see that \(\phi_{X+Y}(t)\) is exactly the MGF of a normal random variable with mean \(\mu_1+\mu_2\) and variance \(\sigma_1^2 + \sigma_2^2\text{.}\) Using Theorem 10.3 (specifically part 1), we see that this means that \(X+Y\) has the distribution of a normal variable with mean \(\mu_1+\mu_2\) and variance \(\sigma_1^2 + \sigma_2^2\text{.}\)
      🔗
      🔗
    Problem 2: You are given an unfair 4-sided die with faces numbered 1, 2, 3, and 4 for which the probability of rolling any particular even number is \(p {\lt} 1/2\) and the probability of rolling any particular odd number is \(1/2-p\text{.}\) Let \(X\) be the number you roll on a toss of this die.
    🔗
    1. Compute \(\phi_X(t)\text{,}\) the moment generating function of \(X\text{.}\)
      🔗
      🔗
    2. Let \(X_1,X_2,\ldots ,X_n\) be the numbers that you roll on \(n\) independent rolls of this die and define \(S_n = X_1 + \cdots + X_n\text{.}\) Compute \(\phi_{S_n}(t)\text{.}\)
      🔗
      🔗
    3. For which values of \(p\) can you find an explicit bound for \(\mathrm{P}(S_n \geq 2.9n)\) that decays exponentially in \(n\text{?}\) Describe how you would find such a bound (but do not compute it explicitly).
      🔗
      🔗
    Solution:
    🔗
    1. Using the definition of moment generating functions, we have
      🔗
      \begin{equation*} \phi_X(t) = \mathrm{E}(e^{tX}) = e^{1t}(1/2-p) + e^{3t}(1/2-p) + e^{2t}p + e^{4t}p = p(e^{2t}+e^{4t}) + (1/2-p)(e^{t}+e^{3t}). \end{equation*}
      🔗
    2. Since \(X_i\) are i.i.d. as \(X\text{,}\) Lemma 10.1 implies
      🔗
      \begin{equation*} \phi_{S_n}(t) = \phi_X(t)^n = (p(e^{2t}+e^{4t}) + (1/2-p)(e^{t}+e^{3t}))^n. \end{equation*}
      🔗
    3. Our goal is to apply Theorem 10.2, the large deviations bound. We have
      🔗
      \begin{equation*} \mathrm{E}(X) = p(2+4) + (1/2-p)(1+3) = 6p + 2 - 4p = 2p+2 = 2(1+p). \end{equation*}
      In order to apply the bound in this case, we need
      🔗
      \begin{equation*} 2.9 {\gt} \mathrm{E}(X) = 2(1+p), \end{equation*}
      which holds if and only if
      🔗
      \begin{equation*} p {\lt} 0.45. \end{equation*}
      Under this condition, we can apply the large deviation bound to conclude that
      🔗
      \begin{equation*} \mathrm{P}(S_n \geq 2.9n) \leq \exp (-nI(2.9)) \end{equation*}
      where \(I(2.9) {\gt} 0\) is the number
      🔗
      \begin{equation*} I(2.9) = \sup \{ 2.9t - \log \phi_X(t) : t {\gt} 0\} . \end{equation*}
      🔗
    Problem 3: Assume that \(T\) is an exponential random variable with \(\mathrm{E}(T) = 1\text{,}\) representing the time a light bulb goes out. Given that \(T = t\text{,}\) the additional time \(X\) it takes a repairman to arrive to fix the bulb is either
    🔗
    1. \(\mathrm{Uniform}(t/2,t)\text{,}\) or
      🔗
      🔗
    2. Exponential with expectation \(1/t\text{.}\)
      🔗
      🔗
    In each case, compute \(\mathrm{E}(X)\text{.}\) In case (b), also compute the conditional density given \(X = x\text{.}\)
    🔗
    Solution:
    🔗
    1. Assuming \(T=t\text{,}\) \(X\) is \(\mathrm{Uniform}([t/2,t])\text{.}\) This means that \(\mathrm{E}(X|T=t) = \frac{3}{4}t\text{.}\) Using the law of total expectation, we can compute
      🔗
      \begin{equation*} \mathrm{E}(X) = \int_0^\infty \mathrm{E}(X|T=t)f_T(t)dt = \frac{3}{4}\int_0^\infty te^{-t}dt = \frac{3}{4}. \end{equation*}
      🔗
    2. Assuming \(T=t\text{,}\) \(X\) is \(\mathrm{Exponential}\) with \(\mathrm{E}(X|T=t) = 1/t\text{.}\) This means \(X\) is \(\mathrm{Exponential}(t)\text{.}\) We can again use the law of total expectation to get
      🔗
      \begin{equation*} \mathrm{E}(X) = \int_0^\infty \mathrm{E}(X|T=t)f_T(t)dt = \int_0^\infty \frac{1}{t}e^{-t}dt = \infty . \end{equation*}
      Now, we need to compute the conditional density of \(T\) given \(X=x\text{.}\) By definition,
      🔗
      \begin{equation*} f_T(t|X=x) = \frac{f_{X,T}(x,t)}{f_X(x)} = \frac{f_{X,T}(x,t)}{\int_0^\infty f_{X,T}(x,t)dt}, \end{equation*}
      so we just need to compute \(f_{X,T}(x,t)\text{.}\) We can use our information about \(f_X(X|T=t)\) for this using the same equation conditioning on \(T\) instead of \(X\text{.}\) Indeed, since \(X\) given \(T=t\) is \(\mathrm{Exponential}(t)\text{,}\)
      🔗
      \begin{equation*} f_{X,T}(x,t) = f_X(x|T=t)f_T(t) = te^{-tx}e^{-t} = te^{-(x+1)t}. \end{equation*}
      We have
      🔗
      \begin{equation*} f_X(x) = \int_0^\infty f_{X,T}(x,t)dt = \int_0^\infty te^{-(x+1)t}dt = \frac{1}{(x+1)^2}. \end{equation*}
      Finally,
      🔗
      \begin{equation*} f_T(t|X=x) = \begin{cases} t(x+1)^2e^{-(x+1)t} & \text{\quad when } 0 \leq t {\lt} \infty \\ 0 & \text{\quad otherwise}. \end{cases} \end{equation*}
      🔗
    Problem 4: Let \(a {\gt} 1\text{.}\) You generate a single number \(U\sim \mathrm{Uniform}([0,1])\) and then independently generate a sequence of numbers \(X_1,X_2,X_3,\ldots \sim \mathrm{Uniform}([0,a])\) until one of them exceeds \(U\text{.}\) Let \(N\) be the number of generations it takes.
    🔗
    1. Compute \(p_N(k)\text{,}\) the pmf of \(N\text{.}\)
      🔗
      🔗
    2. Compute \(\mathrm{E}(N)\text{.}\)
      🔗
      🔗
    Solution:
    🔗
    1. First, suppose that \(U = u\in [0,1]\text{.}\) Then the probability of exceeding \(u\) is \(\frac{a-u}{a}\text{,}\) so \(N\sim \mathrm{Geometric}(\frac{a-u}{a})\text{.}\) Therefore,
      🔗
      \begin{equation*} \mathrm{P}(N=k|U=u) = \bigg(\frac{a-u}{a}\bigg)\bigg(1-\frac{a-u}{a}\bigg)^{k-1} = \bigg(1-\frac{u}{a}\bigg)\bigg(\frac{u}{a}\bigg)^{k-1}. \end{equation*}
      Therefore,
      🔗
      \begin{equation*} \mathrm{P}(N=k) = \int_0^1\mathrm{P}(N=k|U=u)f_U(u)du = \int_0^1\bigg(1-\frac{u}{a}\bigg)\bigg(\frac{u}{a}\bigg)^{k-1}du = \frac{1}{a^{k-1}}\bigg(\frac{1}{k} - \frac{1}{a(k+1)}\bigg). \end{equation*}
      🔗
    2. We can use the law of total expectation. Using the formula for the mean of a geometric variable, we know that \(\mathrm{E}(N|U=u) = (\frac{a-u}{a})^{-1} = \frac{a}{a-u}\text{,}\) so
      🔗
      \begin{equation*} \mathrm{E}(N) = \int_0^1\mathrm{E}(N|U=u)f_U(u)du = \int_0^1\frac{a}{a-u}du = -a\log (a-u)|_0^1 = a\log \bigg(\frac{a}{a-1}\bigg) \end{equation*}
      🔗
