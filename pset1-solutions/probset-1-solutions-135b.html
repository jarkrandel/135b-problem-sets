    Problems Problems

    1.

    Let \(X_1,X_2,X_3,\ldots\) be independent random variables that are all uniform on \([0,1]\text{.}\) Prove that the following sequences of random variables converge in probability and find the constant they converge to. You may use without proof that \(\E(X_i)=\frac{1}{2}\) and \(\var(X_i)=\frac{1}{12}\text{.}\)
    🔗
    1. \(\displaystyle Y_n=\frac{1}{n}\sum_{i=1}^n X_i\)
      🔗
      🔗
    2. \(\displaystyle Y_n=\frac{1}{n}\sum_{i=1}^n X_i^2\)
      🔗
      🔗
    3. \(\displaystyle Y_n=\frac{1}{n}\sum_{i=1}^n X_iX_{i+1}\)
      🔗
      🔗
    Hint for (c). The random variables \(X_iX_{i+1}\) and \(X_{i+1}X_{i+2}\) are not independent, so their covariance is not necessarily \(0\text{.}\) You should be careful when computing \(\var(Y_n)\text{.}\)
    🔗
    Solution (a).
    🔗
    Since \(X_i\) is \(\uniform([0,1])\text{,}\) we know that \(\E X_i=\frac{1}{2}\) for all \(i\) and \(\var(X_i)=\frac{1}{12}\) for all \(i\text{.}\) Since the \(X_i\) are assumed to be independent, they are i.i.d., and we can apply the Weak Law of Large Numbers with \(S_n=X_1+\cdots+X_n\) to conclude that
    🔗
    \begin{equation*} \frac{1}{n}\sum_{i=1}^n X_i \to \frac{1}{2} \end{equation*}
    in probability.
    🔗
    Solution (b).
    🔗
    For any \(i\text{,}\) we have
    🔗
    \begin{equation*} \E(X_i^2)=\int_0^1 x^2\,dx=\frac{1}{3}. \end{equation*}
    Since the \(X_i^2\) are again i.i.d. and their common variance is clearly finite, we can apply the Weak Law of Large Numbers with \(S_n=X_1^2+\cdots+X_n^2\) to conclude
    🔗
    \begin{equation*} \frac{1}{n}\sum_{i=1}^n X_i^2 \to \frac{1}{3} \end{equation*}
    in probability.
    🔗
    Solution (c).
    🔗
    Set \(Z_i=X_iX_{i+1}\text{.}\) We might like to try to apply the Weak Law of Large Numbers as in the previous two parts, but \(Z_i\) and \(Z_{i+1}\) are not independent for any \(i\text{,}\) so we cannot apply the theorem. Let us instead use the more general criterion for convergence in probability from class: since \(X_i\) and \(X_{i+1}\) are independent, we know
    🔗
    \begin{equation*} \E(Z_i)=\E(X_iX_{i+1})=\E(X_i)\E(X_{i+1})=\frac{1}{4}. \end{equation*}
    This means we just have to show that \(\var\!\left(\frac{1}{n}\sum_{i=1}^n Z_i\right)\to 0\text{.}\) Again, the lack of independence makes this a bit more cumbersome since we have to consider covariances of the \(Z_i\text{.}\) That is,
    🔗
    \begin{equation*} \var\!\left(\frac{1}{n}\sum_{i=1}^n Z_i\right)=\frac{1}{n^2}\sum_{i=1}^n \var(Z_i) + \frac{2}{n^2}\sum_{i<j}\cov(Z_i,Z_j). \end{equation*}
    Let us deal with the first term first. Since we already know \(\E(Z_i)\text{,}\) it is convenient to use the formula
    🔗
    \begin{equation*} \var(Z_i)=\E(Z_i^2)-\E(Z_i)^2=\E(Z_i^2)-\frac{1}{16}. \end{equation*}
    Notice that, by independence,
    🔗
    \begin{equation*} \E(Z_i^2)=\E(X_i^2X_{i+1}^2)=\E(X_i^2)\E(X_{i+1}^2)=\frac{1}{3}\cdot\frac{1}{3}=\frac{1}{9}, \end{equation*}
    by part (b), so that
    🔗
    \begin{equation*} \var(Z_i)=\frac{1}{9}-\frac{1}{16}=\frac{7}{144}. \end{equation*}
    Now let us handle the covariance terms. Notice that \(Z_i\) and \(Z_j\) are independent if \(j\) and \(i\) differ by more than \(1\text{,}\) in which case \(\cov(Z_i,Z_j)=0\text{.}\) This means it suffices to consider only the case when \(j=i+1\text{.}\) Recall the formula
    🔗
    \begin{equation*} \cov(X,Y)=\E(XY)-\E(X)\E(Y). \end{equation*}
    Using this, we compute
    🔗
    \begin{equation*} \cov(Z_i,Z_{i+1})=\E(X_iX_{i+1}^2X_{i+2})-\E(Z_i)\E(Z_{i+1}) =\frac{1}{2}\cdot\frac{1}{3}\cdot\frac{1}{2}-\frac{1}{4}\cdot\frac{1}{4} =\frac{1}{48}. \end{equation*}
    We can plug this and our result for \(\var(Z_i)\) into our expression for \(\var\!\left(\frac{1}{n}\sum_{i=1}^n Z_i\right)\) to get
    🔗
    \begin{equation*} \begin{aligned} \var\!\left(\frac{1}{n}\sum_{i=1}^n Z_i\right) &= \frac{1}{n^2}\sum_{i=1}^n \var(Z_i) + \frac{2}{n^2}\sum_{i<j}\cov(Z_i,Z_j) \\ &= \frac{1}{n^2}\frac{7n}{144} + \frac{2}{n^2}\sum_{i=1}^{n-1}\cov(Z_i,Z_{i+1}) \\ &= \frac{1}{n}\frac{7}{144} + \frac{2}{n^2}(n-1)\frac{1}{48} \to 0. \end{aligned} \end{equation*}
    Therefore, we can conclude that
    🔗
    \begin{equation*} \frac{1}{n}\sum_{i=1}^n X_iX_{i+1} \to \frac{1}{4} \end{equation*}
    in probability.

    2.

    You arrange \(n>2\) cubical boxes of side length 1 numbered \(1,2,\ldots,n\) in a circle in clockwise order, independently generate a random number \(X_i\sim\uniform([0,1])\) for each \(1\leq i\leq n\text{,}\) and then fill box \(i\) up to height \(X_i\) with sand. Box \(i\) is a gap box if it is filled at least to height \(3/4\) and the next box in clockwise order is filled to height at most \(1/4\text{.}\) Let \(N_n\) be the number of gap boxes.
    🔗
    1. Define indicator random variables \(I_i\) with \(N_n=I_1+\cdots+I_n\) and compute \(\E(N_n)\text{.}\)
      🔗
      🔗
    2. Use the stated independence to compute \(\var(N_n)\text{.}\)
      🔗
      🔗
    3. Determine a constant \(c\) such that \(\frac{1}{n}N_n\to c\) in probability.
      🔗
      🔗
    Solution (a).
    🔗
    Let
    🔗
    \begin{equation*} I_i = I_{\{\text{box } i \text{ is a gap box}\}}. \end{equation*}
    Then
    🔗
    \begin{equation*} \E(I_i)=\prob\!\left(X_i\geq \frac{3}{4} \text{ and } X_{i+1}\leq \frac{1}{4}\right) =\frac{1}{4}\cdot\frac{1}{4} =\frac{1}{16}. \end{equation*}
    Since \(N_n=I_1+\cdots+I_n\text{,}\), the linearity of expectation means that
    🔗
    \begin{equation*} \E(N_n)=\sum_{i=1}^n \E(I_i)= n\cdot\prob\!\left(\text{box } 1 \text{ is a gap box}\right) = \frac{n}{16}. \end{equation*}
    Solution (b).
    🔗
    First, recall that multiplying indicator random variables produces an indicator random variable that identifies when both constituent events happen:
    🔗
    \begin{equation*} I_iI_j = I_{\{\text{boxes } i \text{ and } j \text{ are gap boxes}\}}, \end{equation*}
    so
    🔗
    \begin{equation*} \E(I_iI_j)=\prob(\text{boxes } i \text{ and } j \text{ are gap boxes}). \end{equation*}
    To compute this probability, first fix \(i\text{.}\) Notice that if box \(j\) is not adjacent to box \(i\text{,}\) then neither box \(j\) nor the next box in clockwise order from it are influenced by the state of box \(i\text{,}\) meaning \(I_i\) and \(I_j\) are independent. However, if \(j=i+1\text{,}\) then
    🔗
    \begin{equation*} \prob(\text{boxes } i \text{ and } i+1 \text{ are gap boxes}) = 0 \end{equation*}
    since this requires both \(X_{i+1} \leq 1/4\) and \(X_{i+1} \geq 3/4\). We also use the index \(n+1\) to refer to box \(1\) and \(n+2\) to refer to box \(2\) for convenience since box \(1\) is clockwise adjacent to box \(n\text{.}\) This means
    🔗
    \begin{equation*} \begin{aligned} \var(N_n) &= \sum_{i=1}^n \var(I_i) + 2\sum_{i<j}\cov(I_i,I_j) \\ &= \sum_{i=1}^n \left(\E(I_i^2)-\E(I_i)^2\right) + 2\sum_{i=1}^n \left(\E(I_iI_{i+1})-\E(I_i)\E(I_{i+1})\right) \\ &= \sum_{i=1}^n \left(\frac{1}{16}-\frac{1}{16^2}\right) + 2\sum_{i=1}^n \left(0-\frac{1}{16^2}\right) \\ &= n\left(\frac{1}{16}-\frac{3}{16^2}\right) = \frac{13n}{256}. \end{aligned} \end{equation*}
    Solution (c).
    🔗
    Notice that
    🔗
    \begin{equation*} \E\!\left(\frac{1}{n}N_n\right)=\frac{1}{n}\cdot \frac{n}{16}=\frac{1}{16} \end{equation*}
    and
    🔗
    \begin{equation*} \var\!\left(\frac{1}{n}N_n\right)=\frac{1}{n^2}\cdot\frac{13n}{256}=\frac{13}{256n}\to 0. \end{equation*}
    Therefore
    🔗
    \begin{equation*} \frac{1}{n}N_n \to \frac{1}{16} \end{equation*}
    in probability, so \(c=\frac{1}{16}\text{.}\)
    🔗

    3.

    Suppose that you have a total of \(2n\) cards consisting of \(n\) pairs of cards where each pair is a different color. Select \(n\) cards without replacement and let \(N_n\) denote the number of colors that are not present in your selection.
    🔗
    1. Compute \(\E(N_n)\text{.}\)
      🔗
      🔗
    2. Compute \(\var(N_n)\text{.}\)
      🔗
      🔗
    3. Find a constant \(c\) so that \(\frac{1}{n}N_n\to c\) in probability. Prove this convergence in probability.
      🔗
      🔗
    Solution (a).
    🔗
    For each \(i\text{,}\) let
    🔗
    \begin{equation*} I_i = I_{\{\text{color } i \text{ is not present}\}} = \begin{cases} 1 & \text{if color } i \text{ is not present in the selection},\\ 0 & \text{otherwise.} \end{cases} \end{equation*}
    Then \(N_n=I_1+\cdots+I_n\text{.}\) By linearity of expectation,
    🔗
    \begin{equation*} \E(N_n)=n\cdot \prob(\text{color } i \text{ is not selected}). \end{equation*}
    Color \(i\) is absent exactly when all \(n\) chosen cards come from the remaining \(2n-2\) cards, so
    🔗
    \begin{equation*} \prob(\text{color } i \text{ is not selected}) =\frac{\binom{2n-2}{n}}{\binom{2n}{n}} =\frac{n-1}{2(2n-1)}. \end{equation*}
    Hence
    🔗
    \begin{equation*} \E(N_n)=\frac{n(n-1)}{2(2n-1)}. \end{equation*}
    Solution (b).
    🔗
    Since these indicator random variables are not independent, we cannot conclude pairwise covariances are zero, so it is more convenient to use
    🔗
    \begin{equation*} \var(N_n)=\E(N_n^2)-\E(N_n)^2. \end{equation*}
    In order to compute \(\E(N_n^2)\text{,}\) notice that
    🔗
    \begin{equation*} N_n^2=(I_1+\cdots+I_n)^2=\sum_{i=1}^n I_i^2+\sum_{i\neq j} I_iI_j =\sum_{i=1}^n I_i+\sum_{i\neq j} I_iI_j, \end{equation*}
    since the random variables \(I_i\) and \(I_i^2\) are identical. Also, \(I_iI_j\) is \(1\) if and only if both colors \(i\) and \(j\) are not selected, and is \(0\) otherwise. This means
    🔗
    \begin{equation*} \E(I_iI_j)=\prob(\text{colors } i \text{ and } j \text{ are not selected}) =\frac{\binom{2n-4}{n}}{\binom{2n}{n}} =\frac{(n-2)(n-3)}{4(2n-1)(2n-3)}. \end{equation*}
    So, finally,
    🔗
    \begin{equation*} \begin{aligned} \var(N_n) &= \E(N_n^2)-\E(N_n)^2 \\ &= \E(N_n)+\sum_{i\neq j}\E(I_iI_j)-\left(\frac{n(n-1)}{2(2n-1)}\right)^2 \\ &= \frac{n(n-1)}{2(2n-1)} + n(n-1)\frac{(n-2)(n-3)}{4(2n-1)(2n-3)} - \left(\frac{n(n-1)}{2(2n-1)}\right)^2. \end{aligned} \end{equation*}
    Solution (c).
    🔗
    Observe that
    🔗
    \begin{equation*} \E\!\left(\frac{1}{n}N_n\right)=\frac{n-1}{2(2n-1)} \to \frac{1}{4}. \end{equation*}
    If we can show that \(\var\!\left(\frac{1}{n}N_n\right)\to 0\text{,}\) then we can apply Theorem 9.1 to prove convergence in probability. We have
    🔗
    \begin{equation*} \begin{aligned} \var\!\left(\frac{1}{n}N_n\right) &= \frac{1}{n^2}\var(N_n) \\ &= \frac{n(n-1)}{2n^2(2n-1)} + \frac{n(n-1)(n-2)(n-3)}{4n^2(2n-1)(2n-3)} - \left(\frac{n(n-1)}{2n(2n-1)}\right)^2 \\ &\to 0 + \frac{1}{16} - \left(\frac{1}{4}\right)^2 = 0. \end{aligned} \end{equation*}
    Thus
    🔗
    \begin{equation*} \frac{1}{n}N_n \to \frac{1}{4} \end{equation*}
    in probability, so \(c=\frac{1}{4}\text{.}\)
