    Problem 1: Find the classes for the Markov chains with the following transition matrices and specify whether each class is transient or recurrent
    🔗
    \begin{equation*} P_1 = \begin{bmatrix} 0 & 1/2 & 1/2 \\ 1/2 & 1/2 & 0 \\ 0 & 1/2 & 1/2 \end{bmatrix}, \quad P_2 = \begin{bmatrix} 1/2 & 1/2 & 0 & 0 \\ 1/3 & 2/3 & 0 & 0 \\ 0 & 0 & 1/2 & 1/2 \\ 1/3 & 0 & 1/3 & 1/3 \end{bmatrix} \end{equation*}
    \begin{equation*} P_3 = \begin{bmatrix} 1/2 & 0 & 1/2 & 0 & 0 \\ 0 & 0 & 1/2 & 0 & 0 \\ 1/3 & 1/3 & 1/3 & 0 & 0 \\ 0 & 0 & 0 & 1/4 & 3/4 \\ 1/10 & 0 & 0 & 0 & 9/10 \end{bmatrix}, \quad P_4 = \begin{bmatrix} 1/2 & 0 & 0 & 0 & 1/2 \\ 0 & 1/2 & 1/2 & 0 & 0 \\ 0 & 0 & 1/3 & 2/3 & 0 \\ 0 & 1/3 & 0 & 1/3 & 1/3 \\ 1/5 & 0 & 0 & 0 & 4/5 \end{bmatrix} \end{equation*}
    Solution: I think the best way to figure this out is just to draw a diagram and chase arrows around. The specific transition probabilities don’t matter; you just need to know whether there is a nonzero probability of transition between the states.
    🔗
    First, determine the classes by finding out which states in the chain communicate with one another. Because these chains are finite, classes are recurrent if and only if they are closed. Check whether they are closed by checking whether any state in the class has nonzero transition probability to another state outside of the class.
    🔗
    1. \(P_1\text{:}\) There is a single class that is recurrent.
      🔗
      🔗
    2. \(P_2\text{:}\) The classes are \(\{ 1,2\} \text{,}\) which is recurrent, and \(\{ 3,4\} \text{,}\) which is transient.
      🔗
      🔗
    3. \(P_3\text{:}\) The classes are \(\{ 1,2,3\} \text{,}\) which is recurrent, \(\{ 4\} \text{,}\) which is transient, and \(\{ 5\} \text{,}\) which is transient (Notice that 4 and 5 do not communicate.).
      🔗
      🔗
    4. \(P_4:\) The classes are \(\{ 1,5\} \text{,}\) which is recurrent, and \(\{ 2,3,4\} \text{,}\) which is transient.
      🔗
      🔗
    Problem 2: A particle can occupy an infinite number of states indexed by the integers \(0,1,2,\ldots \text{.}\) The particle starts at state 0 and transitions at each time step according to one of the following rules: If the particle is in state \(j\text{,}\) then it can move only to either state 0 or state \(j+1\text{.}\) Let’s consider two different cases (1) and (2) for the transition probabilities. In case (1), the particle transitions from \(j\) to \(j+1\) with probability 1/3, while in case (2), it transitions from \(j\) to \(j+1\) with probability \(1-(\frac{1}{j+2})^2\text{.}\)
    🔗
    1. Explain why this chain is irreducible in both case (1) and case (2).
      🔗
      🔗
    2. Let \(R\) be the random variable whose value is the first time the particle returns to 0. Since the particle may not ever return to 0, we let \(R = \infty \) in the case when the particle does not return. Compute \(\mathrm{P}(R {\gt} n)\) for any \(n \geq 1\) in each case. You should assume \(\infty {\gt} n\) so that this final case is included in your calculations.
      🔗
      Hint: In the second case, first show that \(1-(\frac{1}{j+2})^2 = \frac{(j+1)(j+3)}{(j+2)^2}.\) A lot of cancellation should happen in your expression when you take products (use factorials).
      🔗
      🔗
    3. Compute \(f_0\text{,}\) the probability of return to 0 in each case.
      🔗
      🔗
    4. Determine the recurrence or transience of all states in each case.
      🔗
      🔗
    Solution:
    🔗
    1. Let \(i {\lt} j\) represent states. In either case, the transition probability from state \(i\) to state \(i+1\) is positive. This means that we can transition with positive probability from \(i\) to \(i+1\text{,}\) then from \(i+1\) to \(i+2\text{,}\) then ..., then from \(j-1\) to \(j\) so that \(i\rightarrow j\text{.}\) Then, \(j\) can transition to 0 with positive probability and \(0\) can transition to \(j\) with positive probability, showing that \(j\rightarrow i\text{.}\) This shows that any two distinct states communicate.
      🔗
      🔗
    2. In either case, \(R {\gt} n\) if and only if \(R\) is not less than or equal to \(n\) if and only if the chain transitions from \(i\) to \(i+1\) for every \(0 \leq i \leq n-1\) (this is \(n\) time steps). The probability that this happens in case (1) is just
      🔗
      \begin{equation*} \mathrm{P}(R {\gt} n) = \left(\frac{1}{3}\right)^n. \end{equation*}
      In case (2), we get
      🔗
      \begin{equation*} \begin{aligned} \mathrm{P}(R {\gt} n) & = \prod_{j=0}^{n-1}\left(1 - \frac{1}{(j+2)^2}\right) = \prod_{j=0}^{n-1}\left( \frac{(j+1)(j+3)}{(j+2)^2} \right)\\ & = \frac{\left(\prod_{j=0}^{n-1}(j+1)\right)\left(\prod_{j=0}^{n-1}(j+3)\right)}{\left(\prod_{j=0}^{n-1}(j+2)\right)^2}\\ & = \frac{n!((n+2)!/2)}{(n+1)!(n+1)!} = \frac{n+2}{2(n+1)}. \end{aligned} \end{equation*}
      🔗
    3. We need to use the answers from (b). In case (1),
      🔗
      \begin{equation*} 1 - f_0 = \lim_{n\rightarrow \infty }\mathrm{P}(R {\gt} n) = \lim_{n\rightarrow \infty }\left(\frac{1}{3}\right)^n = 0, \end{equation*}
      so \(f_0 = 1\text{.}\) In case (2),
      🔗
      \begin{equation*} 1- f_0 = \lim_{n\rightarrow \infty }\mathrm{P}(R {\gt} n) = \lim_{n\rightarrow \infty } \frac{n+2}{2(n+1)} = \frac{1}{2}, \end{equation*}
      so \(f_0 = \frac{1}{2}\text{.}\)
      🔗
      🔗
    4. Since the chain is irreducible, either all of the states are recurrent or all of the states are transient. This is determined by whether or not \(f_0 {\lt} 1\text{.}\) In case (1), every state is recurrent since \(f_0 = 1\text{,}\) while in case (2) every state is transient since \(f_0 = \frac{1}{2} {\lt} 1\text{.}\)
      🔗
      🔗
    Problem 3: Consider the perfect infinite binary tree in Figure 1. This tree consists of a single root node (at the bottom) and satisfies the following property: Every node in the tree has two child nodes, and any node that is not the root node has a single parent node. Suppose that a random walker moves from node to node in this tree such that at any time the walker is equally likely to transition to any other node connected to its current node
    🔗
    1. Argue that the Markov chain associated to this walk is irreducible.
      🔗
      🔗
    2. We say that a node is at height \(h\) in the tree if a shortest path from the node to the root takes \(h\) steps. Determine whether the chain is recurrent or transient by considering a new Markov chain called the "height chain" that keeps track of the current height of the random walker in the binary tree.
      🔗
      Hint: A walker is at the root if and only if its current height is \(0\text{.}\) You can also cite a result from lecture about gambler’s ruin probabilities to help.
      🔗
      🔗
    Solution:
    🔗
    1. The root is accessible from every node by transitioning to parent nodes repeatedly. Therefore, to get from any node \(i\) to any other node \(j\text{,}\) the chain can transition first from \(i\) to the root, then from the root to \(j\text{.}\) Therefore, \(i\) communicates with \(j\) for any \(i,j\in S\text{.}\)
      🔗
      🔗
    2. Consider the Markov chain on \(S = \{ 0,1,2,3,4,\ldots \} \) where \(P_{01} = 1\) and, for any \(i\not=0\)
      🔗
      \begin{equation*} P_{ij} = \begin{cases} \frac{2}{3} & \text{if $j = i+1$}\\ \frac{1}{3} & \text{if $j = i-1$}\\ 0 & \text{otherwise} \end{cases}. \end{equation*}
      This Markov chain models the height of a random walker in the binary tree, so let’s call it the "height chain". Indeed, if the walker is at height 0, then it is at the root node and it moves to a child node at height 1 with probability 1. If the walker is at height \(h {\gt} 0\text{,}\) then it moves to a child at height \(h + 1\) with probability \(2/3\) and to its parent at height \(h-1\) with probability \(1/3\text{.}\) This chain is a walk on the nonnegative integers with probability \(p = \frac{2}{3}\) of moving to the right. From our analysis of the gambler’s ruin probabilities in lecture, we know that
      🔗
      \begin{equation*} \mathrm{P}(\text{$S_n$ ever reaches $0$ | $S_0 = 1$}) = \frac{1-\frac{2}{3}}{\frac{2}{3}} = \frac{1}{2} {\lt} 1. \end{equation*}
      As soon as the chain enters 0, it goes to \(1\) with probability 1. This means
      🔗
      \begin{equation*} f_0 = \mathrm{P}(\text{chain ever returns to 0}) = \mathrm{P}(\text{$S_n$ ever reaches $0$ | $S_0 = 1$}) = \frac{1}{2} {\lt} 1. \end{equation*}
      Therefore, state 0 is transient in our height chain. This means that the root node is transient in the binary tree because the walker is at the root node if and only if the height chain is in state 0, so the probability of ever returning to 0 in the original chain is the same as the probability of ever returning to 0 in the height chain. This shows the original chain is transient.
      🔗
